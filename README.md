# BCI-HRP Lab PI-Ready Research Materials

This package supplements the BCI-HRP Lab collaboration webpage.

## Files

- `index.html` — PI-ready landing page.
- `protocol_brief.md` — two-page-style protocol brief draft.
- `data_schema.csv` — minimal session data schema.
- `stimulus_sheet_template.csv` — sound probe documentation template.
- `safety_flow.md` — safety interaction artifact.
- `osf_structure.md` — suggested OSF / repository structure.
- `references.bib` — starter BibTeX references.
- `pi_email_variants.md` — PI-specific outreach variants.

## Boundary

This is not an IRB-approved protocol, not a medical system, not treatment, not diagnosis, not crisis detection, and not neural decoding. Partner lab review is required before data collection.


## Pause-QDR preliminary materials

- `pause_qdr_preliminary_dataset.md` — anonymized descriptive summary of the static-pause exploratory dataset.
- `pause_qdr_preliminary_summary.csv` — anonymized derived numeric/categorical summary.
- `pause_qdr_limitations.md` — limitations and correct use.
- `pause_qdr_codebook.md` — preliminary qualitative coding sketch.

Raw participant notes are not included in the public package.


## Language pages

- `index.html` — bilingual gateway.
- `zh.html` — Chinese lab collaboration page.
- `en.html` — English lab collaboration page.


## Data file clarification

Do not confuse `data_schema.csv` with the preliminary static-pause dataset.

- `data_schema.csv` = future protocol data dictionary / field template.
- `pause_qdr_preliminary_summary.csv` = anonymized derived preliminary Pause-QDR data.
- `DATA_FILES_GUIDE.md` explains the difference.
