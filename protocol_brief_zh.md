# BCI-HRP Lab：Sound-QDR 协议简报（中文）

**状态：** 合作草案。未通过 IRB。不是临床、诊断、治疗、危机检测或神经解码协议。

## 1. 研究问题

声音探针 + 自我预测 + 可编辑命名协议，是否能提升个体内自我校准？自主神经/神经代理是否能在自评之外提供额外解释价值？

## 2. 候选假设

**H1：预测校准。** 跨重复 session，参与者的声音前预测误差在个体内下降。

**H2：命名稳定性与颗粒度。** 参与者从泛化情绪词转向更稳定、更细颗粒度的身体—情绪—感知—行动描述。

**H3：代理信号价值。** 可穿戴/神经代理只在部分参与者或声音家族中，在数据质量允许时，为自评之外提供额外解释价值。

## 3. Phase -1：Pause-QDR 静坐暂停前测

在声音探针之前，已收集约 10 人的静坐暂停探索性数据：参与者暂停日常任务约 10–12 分钟，记录前后心情、专注、身体状态与开放式感质笔记。

这不是 Sound-QDR 有效性证据，而是说明未来研究必须设置 pause-only baseline，以区分暂停效应、环境效应、任务需求效应、声音效应和解释自主效应。

## 4. Phase 0 设计

- 个体内可行性试点。
- 非临床参与者，N=5–12。
- N 只用于可用性、数据质量、负面体验发现和后续功效研究方差估计。
- 不做诊断、治疗、危机检测或隐藏情绪推断。

## 5. 单次流程

1. 同意门槛：声音家族、强度上限、停止方式、数据分享、AI 总结许可。
2. 基线：短暂静息 + 自评；可选生理基线。
3. 预测：用户预测声音会如何改变自己。
4. 探针：固定、有文档、有事件标记的音频。
5. 报告：身体、情绪、感知、行动倾向、意义。
6. 重命名：接受、编辑、拒绝、重命名、删除、跳过、停止、落地。

## 6. 安全边界

不是危机系统。高风险表露退出协议，进入合作实验室批准的人类安全流程。
