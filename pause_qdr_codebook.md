# Pause-QDR Qualitative Coding Sketch

This draft codebook can be used for a preliminary qualitative pass. It should be revised with a partner lab.

## Candidate codes

### Attention direction

- environment
- body
- sound
- thought / memory
- task demand
- people / social observation
- future planning
- self-reflection

### Mechanism attribution

- quiet itself
- pause from daily task
- device / phone absence
- environment / context
- breathing / posture
- task importance
- social / relational trigger
- unknown / mixed

### Body-change terms

- relaxation
- tension
- heaviness
- numbness / tingling
- pain / discomfort
- breath / heart rhythm
- posture sensitivity
- no change

### Affective direction

- calm
- pleasure / inspiration
- sadness / loneliness
- anxiety / tension
- emotional flattening
- safety / warmth
- task pressure
- mixed

### Agency events

- wanted to stop
- no stop urge
- phone urge
- task compliance
- observation deepened
- self-reframing
- grounding / settling

## Coding warning

The dataset is too small and too heterogeneous for strong claims. The purpose of coding is to discover protocol-relevant patterns, not to infer clinical or universal effects.
