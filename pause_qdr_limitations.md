# Pause-QDR Preliminary Dataset — Limitations

This dataset is useful as preliminary protocol evidence, not efficacy evidence.

## Main limitations

1. **No control group.** There is no phone-use, ordinary waiting, sound, or no-task comparison.
2. **Environment varies.** Some participants were in cafes, others at home or outdoors.
3. **Duration ambiguity.** Some notes refer to 10 minutes and others to 12 minutes.
4. **Scale ambiguity.** Body-state scoring appears to use lower = more relaxed, but not every participant used the scale consistently.
5. **Incomplete responses.** Some participants did not answer all open-ended questions.
6. **Task-demand effects.** Several participants interpreted the pause as a task and tried to provide a useful answer.
7. **No physiological sensors.** There are no HRV, EDA, respiration, motion, EEG, or fNIRS data in this dataset.
8. **No causal claim.** Mood/focus/body changes cannot be attributed to pausing alone.
9. **Non-clinical boundary.** This is not meditation efficacy evidence, not mental-health treatment evidence, and not a crisis protocol.

## How to use it correctly

Use it as:

- feasibility evidence for pre/post self-report,
- evidence that participants can produce rich qualia notes,
- evidence for designing pause-only baselines,
- evidence for task-demand and familiarity/confound thinking.

Do not use it as:

- proof of BCI-HRP efficacy,
- proof of Sound-QDR efficacy,
- proof that static pausing improves mood,
- clinical or therapeutic evidence.
